This online resource is for documenting work in progress and discussion. The main goal of this work is to propose a portable and modern alternative to the current trigger generation and response capture system used in MEG experiments. The existing setup relies heavily on parallel ports or proprietary hardware.

Main goals:

  1. Generating triggers using an Arduino microcontroller.

  2. Studying the detection of visual stimuli displayed on screen using a photodiode.

  3. Combining both approaches into a unified and functional system.

  4. Clearly documenting the signal circuits involved in MEG experiments to assist future users.
